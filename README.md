## 🌟 特色项目推荐：👾 克雷特龙崛起 (Krayt Rising)

这是一个基于《星球大战》塔图因星系背景深度定制的贪吃蛇动作解谜网页游戏。包含原生 Canvas 粒子渲染、自制 Web Audio 电子音效与智能 AI 寻路竞争系统！

### 🎮 如何开始游戏？
1. 本地双击运行仓库中的 [krayt_rising/index.html](file:///c:/Users/19313/Desktop/All%20my%20files/C++/krayt_rising/index.html) 文件即可在浏览器中开始游玩，**无需安装任何依赖**！
2. 查看游戏的 [详细玩法说明与炼金配方手册 (krayt_rising/README.md)](file:///c:/Users/19313/Desktop/All%20my%20files/C++/krayt_rising/README.md)。

### 🌌 关卡与技能简介
* **Zone 1: 地表洞穴** — 掌握 ⏪ 原力时间回溯（长按 `R` 键倒流时间）。
* **Zone 2: 坍塌隧道** — 地面走过即裂，3 步后化为深渊，玩家与 AI 稍不留神就会坠亡。
* **Zone 3: 重力矿脉** — 按 `G` 键任意翻转重力方向，动态吸引/甩开场上的食物。
* **Zone 4: 生态洞穴** — 苔藓、沃姆鼠、洞穴蛛构成生态食物链，玩家可以捕食蜘蛛获取大成长。
* **Zone 5: 西斯神殿** — 踩准 120 BPM 的节奏警告，避开致命的列/行红色节奏脉冲。
* **Zone 6: 炼金室** — 收集火🔥、水💧、电⚡、暗🌑元素，按 `E` 合成专属护盾、雷暴或冰冻技能。
* **Zone 7: 深渊融合** — 终极考验，前面所有机制在此全部融合！

*💡 **调试秘籍**：游戏中随时按键盘上的数字键 `1` 至 `7`，即可在 7 大关卡之间瞬间传送！*
