# KRAYT RISING | 克雷特龙崛起

🚀 **Krayt Rising** 是一款将经典贪吃蛇玩法与《星球大战》塔图因星系背景深度融合的多机制动作解谜游戏。你将作为一条幼年克雷特龙，深入神秘的沙漠深渊，吞食沃姆鼠、掌握多重原力，最终主宰沙丘！

---

## 🎮 游戏核心特色

* **🌌 塔图因沙海氛围**：写实Q版画风，沙金色鳞片、随动作起伏的暖橙色脊背发光条与生动的龙瞳。洞穴背景包含多日光柱过滤效果、微光颗粒与浮尘粒子，配合利用 Web Audio API 合成的复古电子音效，营造极致沉浸感。
* **⏪ 时间倒流解谜**：长按 **`R` 键**即可进入原力回溯状态。所有的生物位置、分数、食物及关卡进度都将倒流，借此规避致命失误。
* **⚔️ 智能竞争幼龙**：游戏中存在一条深褐色的竞争者 AI 蛇，使用 BFS 寻路与空间洪水填充算法，与你争夺资源。
* **💨 空间跃迁冲刺**：按下 **空格键** 可以朝当前方向瞬间移动 2 格，不仅可用于躲避包抄，还能先发制人抢夺水晶。

---

## 🗺️ 七大深渊区域机制

游戏由 7 个不同主题的地下深渊组成。当你在当前关卡达到目标分数时，地图上会生成旋转的**原力传送门**，走入即可晋级下一关并解锁全新机制。

### Zone 1: 地表洞穴 (Surface Caves)
* **核心机制**：⏪ 原力时间回溯（默认能力）。
* **玩法**：熟悉移动与捕食。吞食凯伯水晶💎能获得双倍成长与特殊音效。

### Zone 2: 坍塌隧道 (Collapsing Tunnels)
* **核心机制**：🕳️ 走过即塌。
* **玩法**：任何蛇（包括你和 AI 竞争者）走过的地面都会开始开裂，并在 3 步之后塌陷为无底深渊。再次踏入深渊（X）会立刻坠落死亡。AI 会聪明地绕开这些坍塌格。

### Zone 3: 重力矿脉 (Gravity Mines)
* **核心机制**：🔄 重力翻转。
* **玩法**：按下 **`G` 键**可以在“下、左、上、右”四个方向之间循环翻转重力。地图上的普通食物、肉块和凯伯水晶会因重力吸引而朝该方向滑动，你可以利用它将远处的食物拉向自己，或者把危险的陷阱推开。

### Zone 4: 生态洞穴 (Living Caves)
* **核心机制**：🧬 动态食物链。
* **玩法**：场景中苔藓会自然生长，沃姆鼠会四处爬行并吃掉苔藓，而紫色的**洞穴蛛**会追逐并捕食沃姆鼠。作为顶端捕食者，你可以吞食沃姆鼠（+15分）或捕食危险的蜘蛛（+25分并成长），但注意不要被蜘蛛咬到头！

### Zone 5: 西斯神殿 (Sith Temple)
* **核心机制**：🎵 节奏脉冲。
* **玩法**：伴随 BPM 120 的低沉警报与红光闪烁，神殿的某些地面会亮起节奏光标。当光标亮起且红光最亮时，若停留在格子上会受到脉冲伤害（蛇身长度 -2）。

### Zone 6: 炼金室 (Alchemist's Lair)
* **核心机制**：🧊 元素合成。
* **玩法**：收集地图上生成的火🔥、水💧、电⚡、暗🌑四大原力元素。收集满 2 个后，按下 **`E` 键**进行炼金合成，施放专属的原力技能（技能持续 40 步）。

### Zone 7: 深渊融合 (The Convergence)
* **核心机制**：⚠️ 全部机制大融合。
* **玩法**：无尽挑战关卡。塌陷地面、重力翻转、生态捕食、节奏脉冲和元素合成将同时生效，测试你的原力极限！

## 🔮 炼金合成公式一览 (Zone 6 & 7)

收集两个元素并按下 **`E`** 可以合成以下技能：

| 元素配方 | 技能名称 | 视觉效果 | 技能作用 |
|---|---|---|---|
| **火🔥 + 暗🌑** | **原力腐蚀 (Corrosion)** | 焦黄色光环 | 产生腐蚀反应。 |
| **水💧 + 暗🌑** | **暗潮 (Dark Tide)** | 幽蓝色微光 | 减缓竞争对手 AI 蛇的移动速度。 |
| **电⚡ + 暗🌑** | **原力风暴 (Force Storm)** | 紫色电光 | 释放雷击，立刻清空全屏的所有蜘蛛。 |
| **火🔥 + 水💧** | **蒸汽护盾 (Steam Shield)** | 淡蓝色护盾 | 获得一次免死机会，抵消一次撞墙、撞蛇或坠入深渊的伤害。 |
| **火🔥 + 电⚡** | **等离子爆 (Plasma Blast)** | 亮橙色爆破 | 释放瞬间能量脉冲。 |
| **电⚡ + 水💧** | **原力冻结 (Freeze)** | 冰蓝色冻结 | 瞬间冻结竞争对手，使其无法移动数秒。 |

## ⌨️ 键位操作指南

* **方向键 / WASD**：上下左右移动
* **空格键 (Space)**：原力瞬移冲刺（CD 4秒）
* **`R` 键 (长按)**：激活原力回溯，松开则恢复正常时间流速
* **`G` 键**：顺时针翻转重力方向（限 Zone 3, 7）
* **`E` 键**：合成当前收集的两个元素（限 Zone 6, 7）
* **`1` ~ `7` 数字键（调试秘籍）**：随时按下可直接跳关至 Zone 1 到 Zone 7，方便单独测试各个关卡机制

## 🛠️ 技术栈与实现

1. **核心逻辑**：纯 JavaScript 编写的无依赖单文件网页游戏，Canvas 图形渲染与 Game Loop 控制。
2. **AI 系统**：使用 BFS (广度优先搜索) 进行目标追踪寻路，结合 Flood Fill (洪水填充算法) 评估行进路线的安全空间大小，保证 AI 在复杂坍塌地形中也能自如穿梭。
3. **音频系统**：基于 Web Audio API 动态合成的声效系统（Sine/Square/Sawtooth 波高频转换），免去外部音频文件的加载延迟。
4. **状态机系统**：通过状态快照栈（Snapshot Stack）实现毫秒级的状态回滚，保证回溯的平滑与精准。
